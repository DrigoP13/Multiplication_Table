Multiplication Tips:

NUMBERS ENDING IN 0


In order to multiply by a number ending in zero, we first need to multiply the number as if it didn’t have zeros, and after, we just add all of the zeros the number had to the final answer.

Look:


23 x 10 = (23 x 1) = 23 → Add the the amount of zeros at the end: 23 → 230

35 x 300 = (35 x 3) = 105 → Add the the amount of zeros at the end: 105 → 10500

12 x 2000 = (12 x 2) = 24 → Add the the amount of zeros at the end: 24 → 24000

MULTIPLYING BY 5


Multiplying a number by 5 means to multiply it by 10 and then divide it by 2 (since 5 = 10/2). Therefore, we are gonna use the last tip, adding a 0 to the number and getting the half of it.

Look:


17 x 5 = (17 x 10) = 170 → Divide the result by 2: 170/2 = 85

35 x 5 = (35 x 10) = 350 → Divide the result by 2: 350/2 = 175

94 x 5 = (94 x 10) = 940 → Divide the result by 2: 940/2 = 470

MULTIPLYING BY 9


From the same line of reasoning of the last tip, we can consider 9 as (10 - 1). This will lead us to conclude that multiplying a number by 9 is the same that adding a 0 on it and subtract the inicial number from it.

Look:


85 x 9 = (85 x 10) = 850 → Subtract 85 from the result: 850 - 85 = 765

154 x 9 = (154 x 10) = 1540 → Subtract 154 from the result: 1540 - 154 = 1386

52 x 9 = (52 x 10) = 520 → Subtract 52 from the result: 520 - 52 = 468

MULTIPLYING BY 11


(This tip can only be applied when the number we will multiply has 2 digits) Whenever you need to multiply a number by 11, simply add its digits and place the result between them.

Look:


26 x 11 = (2 + 6) = 8 → Put the 8 between the 2 digits: 2 8 8 = 288

43 x 11 = (4 + 3) = 7 → Put the 7 between the 2 digits: 4 7 3 = 473

90 x 11 = (9 + 0) = 9 → Put the 9 between the 2 digits: 9 9 0 = 990

MULTIPLYING BY 101


An interesting pattern can be noticed when multiplying a two digit number by 101. You just need to write down the number twice.

Look:


99 x 101 = → Just write the number again: 9999

26 x 101 = → Just write the number again: 2626

78 x 101 = → Just write the number again: 7878

GENERAL TIP


If none of these tricks can be used in your multiplication, you can still make use of the decomposition technique. Decompose one of the numbers by writing it as an addition of its different parts, and then multiply the other number by these parts.

Look:


45 x 87 = 45 * (80 + 7) → Multiply separadly and add the results: 3600 + 315 = 3915

72 x 14 = 72 * (10 + 4) → (72 * 10) + (72 * 4) = 720 + 288 = 1008

23 x 132 = 23 * (100 + 30 + 2) → (23 * 100) + (23 * 30) + (23 * 2) = 2300 + 690 + 46 = 3036
